OptKit is a tool to help you optimize any website for more conversions. 

== Description ==

Understand and react to your visitors behavior in real-time. You can design any creative you or your team imagines, and deliver it to exactly who you want and when you want.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "OptKit v3.0" settings configure the your Website ID.
4. Log into your OptKit Dashboard: https://go.optkit.com
